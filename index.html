Toon De Bie

Ik ben Toon De Bie. Ik ben geboren op 2 augustus 2001 en wat mij op de dag van vandaag 18 jaar oud maakt. Ik woon in Merksplas samen met mijn ouders en mijn broer. Op Thomas More Geel volg ik sinds 2019 de opleiding IT Factory. Een sprong in het diepe aangezien ik nog maar weinig in aanraking ben gekomen met het echte IT leven. Het middelbaar heb ik voltooid in de richting economie-wiskunde op het Klein Seminarie in Hoogstraten. Voorkennis had ik dus totaal niet maar toch heeft mijn interesse voor technologie, programmeren, etc mij overhaald om hier te starten.

Kijk gerust even rond naar wat ik dit jaar allemaal heb gepresteerd als eerstejaars IT Factory student!

Realisaties

Professionele Ontwikkeling

Een vak waarvoor je ofwel slaagt ofwel niet... Bewijzen wat mijn ontwikkeling was het voorbije jaar was de taak door middel van het tonen van projecten en realisaties. Een portfolio van een eerstejaars IT Factory student.

Internet Of Things

Mijn project de glazenwasser was mijn pronkstuk voor dit vak. Met behulp van een Raspberry Pi heb ik met dit project een potentiemeter, ultrasoonsensor, stappenmotor en LCD-scherm kunnen aansturen.

Windows

Wil je weten of jouw computer voldoet aan bepaalde voorwaarden? Dan zijn pester infrastructure tests the way to go! Deze test kijkt o.a. je opslag, .NET versie en firewall instellingen na.

Java

Van buitenaf een heel simpel programmatje om je supermarktketen mee te kunnen runnen. Binnenin een doolhof van java-code die jou de mogelijkheid geeft alles vlotjes in te stellen!
